Overview

Purpose

I want to build a website about camping, and people can look thorough this website to pick some camping site that they like, and buy the ticket on it.

Audience

The audience is mainly for college students who are excited to approach to the nature.

Dynamic elements

My site will include multiple JavaScript dynamic features:

1. A searchable campground list where users can type keywords to find different camping activities.
2. A BGM (background music) player that can play or pause relaxing nature sounds when the user clicks the icon.
3. A custom select box for choosing different payment methods when booking a ticket.
4. DOM interaction, event listeners, conditional logic, and array methods will be used to filter, sort, and display campgrounds dynamically.
